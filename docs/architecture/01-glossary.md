# 01. Glossary

- **Page** — логическая страница вики.
- **Revision** — неизменяемая версия содержимого страницы (append-only).
- **Head revision** — текущая “актуальная” ревизия страницы.
- **Read-model** — денормализованное представление для быстрых чтений (HTML snapshot, мета, ссылки).
- **Outbox** — коллекция событий/задач, записываемая в одной транзакции с бизнес-изменением.
- **Polling worker** — процесс, который периодически забирает outbox-записи по lease, обрабатывает и отмечает завершение.
- **Lease** — временная “аренда” записи outbox конкретным воркером, чтобы избежать конкуренции.
- **Idempotency-Key** — ключ идемпотентности для write-операций (защита от дублей при ретраях).
- **JWKS** — JSON Web Key Set (публичные ключи для проверки JWT).
- **SLO** — целевые показатели качества сервиса (латентность, доступность).

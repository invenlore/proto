# 18. Testing

## Контрактные тесты
- `proto` — источник истины: генерация должна быть воспроизводимой.
- Для gateway: проверка, что все HTTP маршруты зарегистрированы и отвечают.

## Интеграционные тесты
- docker-compose: mongo + redis + сервис
- Проверка:
  - login → JWT → доступ к protected endpoint
  - outbox: запись события → worker обработал → read-model обновился

## Нагрузочные тесты
- Основной сценарий: GetPage (горячий)
- Проверка p95/p99, error rate, saturation

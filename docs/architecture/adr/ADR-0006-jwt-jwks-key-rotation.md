# ADR-0006: JWT + JWKS + key rotation

## Статус
Accepted (2026-01-17)

## Контекст
Gateway должен валидировать токены без вызова identity на каждый запрос.

## Решение
identity публикует JWKS. Gateway кэширует JWKS и валидирует JWT локально. Ключи ротируются через `kid`.

## Последствия
+ Быстрый auth на read-пути
- Нужно следить за clock skew и ротацией

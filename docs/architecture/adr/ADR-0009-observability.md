# ADR-0009: Observability stack

## Статус
Accepted (2026-01-17)

## Контекст
Уже используются Prometheus+Grafana и FluentBit. Нужно стандартизировать метрики/логи.

## Решение
Все сервисы отдают `/metrics`, пишут структурные JSON логи, request_id сквозной.

## Последствия
+ Единая диагностика
- Требуются общие библиотеки/interceptors

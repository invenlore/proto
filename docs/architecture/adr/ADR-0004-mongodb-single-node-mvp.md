# ADR-0004: MongoDB single-node for MVP

## Статус
Accepted (2026-01-17)

## Контекст
Нужен быстрый MVP с минимальным операционным усложнением.

## Решение
Используем MongoDB в режиме single node (рекомендуется 1-node replica set для транзакций).

## Последствия
+ Быстрый старт
- Ограниченная отказоустойчивость
- Требовательность к индексам и размерам документов
